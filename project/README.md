# First website
#### Video Demo:  <https://youtu.be/Jq66i7tHqDM>
🧩 Description

First Website is a personal web project that introduces who I am and documents my 3-month coding journey over the summer. It is a multi-page portfolio built using HTML, CSS, JavaScript, Node.js, Python, and React Native, with each technology serving a unique role in the project.

The main goal was to design a simple yet functional personal website — one that showcases my learning process, highlights my projects, and integrates basic interactive web features such as dark mode and password-protected pages.

⚙️ Technical Overview
Languages & Tools

Frontend: HTML, CSS, JavaScript, Java

Backend / Conversion: Python (converted into JavaScript for browser compatibility)

Frameworks: Bootstrap (for layout, dark mode, and icon styling)

AI Tools Used: Ninja AI (for code improvement)

The homepage/ directory contains all primary pages and subprojects — from my learning reflections to interactive web games and AI experiments.

🏠 Website Sections
1. Home (index.html)

The landing page provides:

A brief introduction about myself and my motivation for building this website.

A “Read My Story” button leading to the About page.

A Search Bar that lets users find words across all pages.

A Dark Mode toggle for visual comfort.

A Milestones section linking to major project areas:

CS50.html — my learning experience in Harvard’s CS50x course.

/game/ — browser mini-games I developed.

/AI/ — pages exploring Artificial Intelligence and Machine Learning.

Each of these milestones represents an important stage in my learning journey.

📖 About Section
2. About (about.html)

The About page introduces my background, family, and studies.
For privacy reasons, personal information is limited.

It also contains sections linking to my coding journey through:

CS50.html: an overview of Harvard’s CS50x, how I discovered it, and my personal reflections.

Linked zip files of my completed problem sets.

References to course instructors and resources that inspired me (including a tribute to David Malan, the CS50x lecturer).

💻 Contact & Hobbies
3. Contact (contact.html)

This section is password-protected.

Users must enter the correct password (123) to access my social media links, GitHub, and personal notes.

Wrong passwords trigger an alert message.

Successful access redirects users to password.html, which provides all contact details.

4. Hobbies (hobbies.html → me.html)

Also password-protected with the same password (123) for simplicity.

Once authenticated, users can explore me.html, which displays my interests in art, music, films, games, and more.

The authentication system is shared with the Contact page, meaning one correct login unlocks both.

These files represent my early exploration of AI integration within personal projects and Python-to-JS adaptation.

🎮 Interactive & Visual Features

Each HTML file ranges from 200–1000 lines of code.

Includes a custom word search that navigates keywords across pages.

Dark Mode toggle dynamically switches between light and dark themes.

Light Mode Design:

Gradient from pink (top) to purple (bottom), symbolizing dawn — light entering a quiet room.

Dark Mode Design:

Gradient reversed (purple top, pink bottom) to resemble sunset and evening family time.

CSS Styling:

styles.css ensures consistent layout, responsive design, and aesthetic color transitions.
