Started with Python basics and discovered the fascinating world of data science. Completed my first online course on machine learning fundamentals and built simple linear regression models.

March 2023

Deep Dive into Algorithms

Explored various ML algorithms including decision trees, random forests, and SVM. Started working with real datasets and learned about data preprocessing, feature engineering, and model evaluation techniques.

Neural Networks & Deep Learning

Began studying neural networks and deep learning concepts. Implemented my first neural network from scratch and started experimenting with TensorFlow and PyTorch for more complex architectures.

Discovered the power of Natural Language Processing and Large Language Models. Started experimenting with transformer architectures, BERT, and GPT models. Built my first chatbot using pre-trained models.

Explored fine-tuning techniques, prompt engineering, and RAG (Retrieval-Augmented Generation) systems. Started building more sophisticated applications using OpenAI's API and open-source alternatives like Llama and Mistral.
